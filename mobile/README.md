# 📱 Catetin Mobile

The mobile application for **Catetin – Smart AI-Powered Financial Tracker**, built for Android to provide on-the-go financial tracking and AI-powered scanning.

---

## 🛠 Tech Stack

- **Language:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Networking:** Retrofit / Ktor
- **Local DB:** Room
- **DI:** Hilt / Koin

---

## 🧩 Features (Planned)

- **📸 Receipt Scanner:** Capture photos of receipts for automatic entry.
- **🍱 Food AI:** Scan your meal to estimate cost and category.
- **🔔 Smart Notifications:** Reminders and daily financial summaries.
- **📊 Interactive Charts:** Visualize your spending patterns on mobile.

## ⚙️ Getting Started

### 1. Prerequisites
- Android Studio Ladybug or newer
- JDK 17+
- Android SDK 24+ (Android 7.0+)

### 2. Build and Run
1. Open the `mobile` folder in Android Studio.
2. Sync Project with Gradle Files.
3. Run on an Emulator or Physical Device.
