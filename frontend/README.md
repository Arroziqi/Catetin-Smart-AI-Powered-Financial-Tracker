# 🌐 Catetin Frontend

The web dashboard for **Catetin – Smart AI-Powered Financial Tracker**. A beautiful, responsive interface to manage your finances and view AI insights.

---

## 🛠 Tech Stack

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Components:** Radix UI / Shadcn UI (Planned)
- **State Management:** React Context / Zustand (Planned)

---

## 🧩 Features

- **📊 Financial Dashboard:** Real-time summary of income and expenses.
- **💸 Transaction Management:** List, filter, and manually add transactions.
- **💡 AI Insights View:** Narrative-style financial advice generated by AI.
- **📱 Responsive Design:** Optimized for mobile, tablet, and desktop browsers.

---

## ⚙️ Getting Started

### 1. Prerequisites
- Node.js 18.x or higher
- npm / pnpm / yarn

### 2. Install Dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to see the application.
